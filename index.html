Features

Everything your agent needs, nothing it doesn't.

40+ Tools

Comprehensive toolkit covering memory, search, file ops, and more, all from one server.

Agent Memory

Persistent memory storage lets agents recall context across sessions and conversations.

Web Fetch

Retrieve and parse web content with built-in extraction. Clean text, ready for context.

Screenshots

Capture page snapshots for visual context. Useful for debugging and verification.

Multi-Agent Coordination

Coordinate multiple agents through shared state and messaging primitives.

Stdio Transport

Standard stdio transport with stdlib JSON-RPC. No HTTP servers, no extra deps.

Single Binary

One Go binary. No runtime, no containers, no orchestration. Just install and run.

Zero Config

Sensible defaults out of the box. Get started in seconds, configure later if needed.

Go Performance

Built in Go for speed and reliability. Minimal footprint, maximal throughput.

Open source, open access. MIT licensed, fully transparent. Every line of Relay is yours to study, fork, and extend.
