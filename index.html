the problem
F I V E S E R V E R S . F I V E H E A D A C H E S .
file server + image server + pdf server + web fetch server + data server

each one has its own install.
each one has its own config.
each one has its own update cycle.
each one has its own way of breaking.

relay is one binary with 40 tools across 7 categories.
one install. one config. one process.
when it breaks, you know which thing broke.
