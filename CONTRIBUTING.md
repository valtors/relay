# Contributing to Relay

Thanks for your interest in Relay. This is a small project maintained by a solo builder.

## Ways to help

- Star the repo and share it with others
- Open an issue if you find a bug
- Suggest features via GitHub Discussions
- Submit PRs for small fixes

## How to run locally

1. Clone the repo
2. Install Go 1.21+
3. Run `go build -o relay .`
4. For the npm package, see `npm/` directory

## Code style

- Go stdlib first. Avoid external deps unless necessary.
- MIT licensed. Keep it simple.
